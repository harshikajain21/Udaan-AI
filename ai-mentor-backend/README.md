# AI Mentor Backend (Decision Intelligence Engine)

An AI-powered academic and career decision system.

## Features
- Class 10 Stream Intelligence
- Class 12 Degree & Career Intelligence
- Institutional Matching
- Decision Intelligence Scoring
- Risk Analysis
- Improvement Roadmap
- What-If Simulation Engine

## Run Locally

1. Create virtual environment
2. Install dependencies:

pip install -r requirements.txt

3. Run server:

uvicorn main:app --reload

4. Open:

http://127.0.0.1:8000/docs
